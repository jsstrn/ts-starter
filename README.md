# TypeScript starter code

This is starter code for a new TypeScript project.

Run tests

```bash
npm test
```

Run tests in watch mode

```bash
npm run test:watch
```

Build the project

```bash
npm run build
```

Start the project

```bash
npm start
```

Start the project in development mode

```bash
npm run start:dev
```

Start the project in development mode and watch for changes

```bash
npm run start:watch
```
